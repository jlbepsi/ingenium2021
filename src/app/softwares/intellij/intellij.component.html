La suite Intellij

Vous pouvez avoir tous les logiciels JetBrain (PHPStorm, WebStorm, Intellij, ...) avec votre adresse mail EPSI. Pour cela vous devez faire quelques manipulations:
  1. Enregistrez-vous :
    Allez sur la page d'enregistrement (indiquez bien votre adresse mail @epsi.fr)

  2. Validez le mail de JetBrain :
    Lorsque vous recevez le mail de JetBrain, validez-le pour pouvoir entrer votre mot de passe.

  3. Attendez la confirmation de JetBrain :
    Vous devriez maintenant pouvoir utiliser tous les logiciels de JetBrain. Votre licence est valide 1 an.

  4. Renouvellement de la licence :
    Quelques jours avant la date d'expiration de votre licence, vous recevrez un mail de jetBrain vous demandant de la renouveller.
